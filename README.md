# 🎮 游戏用户留存分析项目（Mobile Game Retention Analysis）

## 📌 项目简介

本项目基于移动游戏用户行为数据，对用户留存与流失情况进行分析，重点关注用户生命周期与留存表现，为游戏运营与用户增长提供数据支持。

---

## 📊 分析内容

* 计算 D1、D7、D30 留存率
* 构建 Cohort 分析用户留存变化
* 分析用户活跃天数分布
* 识别用户流失特征

---

## 🔍 核心发现

* D1 留存约 82%，但 D7（约24%）和 D30（约0.2%）下降明显，呈现“高开低走”特征
* 大部分用户仅活跃 1–3 天，用户生命周期较短
* 不同批次用户留存趋势一致，说明流失问题主要来源于产品机制

---

## 💡 业务建议

* 优化新手引导，降低初期流失
* 在第2-3天增加激励机制（任务、奖励等）
* 增强中长期内容设计，提高用户粘性
